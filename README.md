# Виджет банковских операций

Виджет для отображения последних успешных банковских операций клиента с возможностью фильтрации, сортировки и маскировки конфиденциальных данных.

## Описание проекта

Проект представляет собой Python‑библиотеку для работы с банковскими операциями. Основные возможности:

* маскировка номеров банковских карт и счетов;
* фильтрация операций по статусу (выполненные, отменённые и т. д.);
* сортировка операций по дате (по возрастанию/убыванию);
* преобразование форматов даты;
* обработка данных в едином интерфейсе.

## Создайте и активируйте виртуальное окружение:
**Linux/Mac:** ppython3 -m venv venv
source venv/bin/activate

**Windows:** py -m venv venv
venv\Scripts\activate

**Установите зависимости:** В проекте используется pyproject.toml для управления зависимостями. Установите их с помощью pip: pip install -e .

-e (editable mode) — позволяет вносить изменения в код и сразу видеть их эффект без переустановки пакета.

Либо, если у вас установлен Poetry, используйте: poetry install

**Проверьте установку:** Запустите простой Python‑скрипт, чтобы убедиться, что модули импортируются корректно:
python -c "from src.masks import get_mask_card_number; print('Установка прошла успешно!')"

Если вы видите сообщение «Установка прошла успешно!», значит всё настроено правильно.

**Опционально: установите инструменты разработки** Для удобства разработки установите линтеры и форматировщики кода:pip install black flake8 isort mypy
Или через Poetry:
poetry install --with dev
