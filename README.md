# Виджет банковских операций

Виджет для отображения последних успешных банковских операций клиента с возможностью фильтрации, сортировки и маскировки конфиденциальных данных.

## Описание проекта

Проект представляет собой Python‑библиотеку для работы с банковскими операциями. Основные возможности:

* маскировка номеров банковских карт и счетов;
* фильтрация операций по статусу (выполненные, отменённые и т. д.);
* сортировка операций по дате (по возрастанию/убыванию);
* преобразование форматов даты;
* обработка данных в едином интерфейсе.

### Особенности реализации

- **Маскирование карт**: первые 4 и последние 4 цифры видны, остальное скрыто (`********`).
- **Маскирование счетов**: первые 5 и последние 4 цифры видны, между ними 10 звёздочек (согласно требованиям тестов).
- **Обработка дат**: ISO‑строки конвертируются в формат `ДД.ММ.ГГГГ ЧЧ:ММ:СС` с **округлением секунд** (микросекунды > 500 округляются вверх).
- **Устойчивость**: все функции корректно обрабатывают `None`, пустые строки и некорректные данные, возвращая `None` вместо исключений.
